Minimum PHP Requirements

5.4.0

  • Shortened syntax for declaring arrays (> 5.4.0).
  • For the MySQLi 'connect_error' property used in the connect() method in /php/classes/DB.class.php (> 5.3.0).
  • Anonymous autoloader function /php/autoloader.php (> 5.3.0).
  • Usage of the __DIR__ magic constant when requiring or including files (> 5.3.0).

Base HREF

The base HREF is set to <base href="/dsa-oop-web-app-assignment/"> because no virtual host is used.
dsa-oop-web-app-assignment is a directory inside www, using WAMP the full path is C:\wamp\www\dsa-oop-web-app-assignment\ so the app expects to be in a folder called dsa-oop-web-app-assignment for relative links to work from page to page.
If the relative links are not working inside the app just edit or remove the base HREF directive in /tpl/inc/head-section.inc.php

Database Setup

The database setup script /tasks-setup.sql creates the dsa_oop_web_app_assignment database & imports some sample tasks into the tasks table.

Coding Style

PHP code is writting using the PSR-2 Coding Style Guide - https://github.com/php-fig/fig-standards/blob/master/accepted/PSR-2-coding-style-guide.md.

Class Autoloader

Instead of requiring or including files for dependent class usage - the anonymous autoloader function /php/autoloader.php loads dependent classes.

Database Classes

The core database class /php/classes/DB.class.php establishes the database connection with credentials retrieved from /php/dbConn.inc.php.

This database class is used for every database query with each child class extending this class.

Each DB child class follows the same naming convention {table_name}.class.php and is named according to the table it interacts with.

Every query to the database is performed using prepared statements with the MySQLi library (executed through the core DB class) for security (anti SQL Injection) & caching performance.

HTML/CSS

HTML/CSS is enhanced using the Bootstrap front-end framework.

Tasks - CRUD Functions

Create a Task

task/create.php calls Tasks class->create method - Create a new task in the tasks table.

  • Create a new task: task/create.php

View an Individual Task

task/view-task.php calls Tasks class->get method - Retrieves a single task from the tasks table based on the task-id query string.

  • View existing task: task/view-task.php?task-id=1
  • View a NON existing task: task/view-task.php?task-id=1000
  • View an existing task but do not supply a task id: task/view-task.php?task-id or task/view-task.php

View All Tasks

task/view-all-tasks.php calls Tasks class->getAllTasks method - View all tasks in the tasks table.

  • task/view-all-tasks.php

Edit a Task

task/edit.php calls Tasks class->update method - Edit an existing task in the tasks table based on the task-id query string.

  • Edit a task: task/edit.php?task-id=1
  • Attempt to edit a NON existing task: task/edit.php?task-id=1000
  • Edit an existing task but do not supply a task id: task/edit.php?task-id or task/edit.php

Delete a Task

task/delete.php calls Tasks class->delete method - Delete an existing task in the tasks table based on the task-id query string.

  • Delete a task: task/delete.php?task-id=2
  • Attempt to edit a NON existing task: task/delete.php?task-id=1000
  • Edit an existing task but do not supply a task id: task/delete.php?task-id or task/delete.php
